# Ponto A

Este repositório reúne duas versões do site Ponto A.

- `current-live-static/`: backup estático da versão publicada em `opontoa.com` neste momento.
- `source/`: fonte editável do Ponto A para futuras alterações.

Os diretórios de dependências, builds, caches e arquivos de ambiente são intencionalmente ignorados.
