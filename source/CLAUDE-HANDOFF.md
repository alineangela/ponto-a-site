# Project handoff

## Main files

- `app/page.tsx`: page structure, interactions, scroll effects, grain canvas, FAQ, hotspots and confetti physics.
- `app/globals.css`: all primary site styling and responsive rules.
- `public/`: every image and processed transparent PNG used by the prototype.
- `scripts/`: Windows-compatible build and local preview helpers.

## Run locally

```powershell
npm.cmd install
npm.cmd run build
npm.cmd run preview
```

Open `http://localhost:4176/`.

## Notes

- Fonts: Instrument Serif for titles and Inter for body text.
- The dynamic grain is generated by the `NoiseOverlay` canvas in `app/page.tsx`.
- The project uses React, vinext and Matter.js.
- This is an evolving visual prototype. Preserve the numbered section labels while revising individual sections.
